# Prenatal Activity Tracker

IB Computer Science Internal Assessment

A mobile application for prenatal patients and their clinicians.

## Tech Stack
- React Native + Expo (managed workflow)
- JavaScript ES6+ (no TypeScript)
- Firebase Authentication (email + password)
- Cloud Firestore
- AsyncStorage (offline support)
- React Context API
- React Navigation v6
- Jest (unit testing)

## Getting Started
1. Copy `firebase.example.js` to `firebase.js` and fill in your Firebase config.
2. Run `npm install` to install dependencies.
3. Run `npx expo start` to launch the development server.
